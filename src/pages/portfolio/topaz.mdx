# Topaz

Topaz is an Angular component library I built for Pearson VUE, giving their products a consistent, reusable interface instead of every team rebuilding the same patterns from scratch.

## Built on Angular Material

Topaz wraps Angular Material into a customizable set of components — everything from basic buttons and inputs to more complex data tables and modals — all easy to drop into any Pearson VUE product without a team having to reinvent the pattern themselves.

## Accessibility First

The hardest part was accessibility — making sure every component held up with screen readers and keyboard navigation, not just visually. I tested and tuned each component individually rather than treating accessibility as a final pass over the finished library.

## Working with Design

I worked closely with Pearson VUE's design team throughout, so the library ended up reflecting their visual language rather than mine — the goal was a library other teams would actually want to adopt, not one they'd fight against.
