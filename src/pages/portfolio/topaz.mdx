# Topaz

Topaz is an Angular component library I built to give Pearson VUE's products a consistent, reusable interface instead of every team rebuilding the same patterns from scratch.

It's built on Angular Material — a customizable set of components ranging from basic buttons and inputs to more complex data tables and modals, all easy to drop into any project.

The hardest part was accessibility — making sure every component held up with screen readers and keyboard navigation, not just visually. I tested and tuned each one individually rather than treating accessibility as a final pass.

I worked closely with Pearson VUE's design team throughout, so the library ended up reflecting their visual language rather than mine.
